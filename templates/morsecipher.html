Morse Code



Morse Code was created by Samuel Morse to transmit letters across telegrams. He wanted frequently used letters to have short codes and less frequently used letters to have longer codes.
When encrypting, only letters and numbers will be encoded and the rest will be treated like spaces. When decrypting, only periods and hyphens will be decoded and the rest will be treated like spaces.




How to use the above tool?

Enter the plaintext to encrypt in morse code or enter morse code to decrypt it according to the following given morse codes.
